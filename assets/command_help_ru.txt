Command syntax:
текст - писать точно так же
<аргумент: тип> - обязательный аргумент, заменить требуемым значением нужного типа
[аргумент: тип] - необязательный аргумент
a|b - выбрать либо a, либо b

Commands:
score add <игрок: целое> <кол-во: целое> - добавить очков игроку
score add all <кол-во: целое> [... <кол-во: целое>] - добавить очки всем игрокам сразу
score remove <игрок: целое> <кол-во: целое> - отнять очки у игрока
score remove all <кол-во: целое> [... <кол-во: целое>] - отнять очки у всех игроков сразу
score set <игрок: целое> <кол-во: целое> - поставить конкретное кол-во очков игроку
score limit set <значение: целое> - установить лимит очков
score limit mode <least|most> - побеждает тот, у кого меньше всего или больше всего очков
double <0|6> <игрок: целое> - игрок закончил дублем
rename <игрок: целое> <имя: текст> - изменить имя игрока
rename game <имя: текст> - переименовать игру
menu [не_сохранять] - сохранить (если не_сохранять не указан) и выйти в меню
file set [путь_к_файлу: текст] - изменить файл сохранения
file reload - перезагрузить файл сохранения
file save - сохранить игру
log add <что-нибудь: текст> - добавить что-нибудь в журнал
log remove <от: целое> [до: целое] - удалить что-то из журнала
goal <игрок: целое> fixed <значение: целое> - установить цель игрока равной <значение> побед
goal <игрок: целое> relative <игрок: целое|most|least> [<действие: +|-|/|*> <число: дробное>] - установить цель игрока относительно другого
victory <игрок: целое> <значение: целое> - установить кол-во побед игрока
help - получить помощь по командам
